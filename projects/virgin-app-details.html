Project Overview

The Virgin App was developed during the StartHack 2025 hackathon in Switzerland, where our team was challenged to create a unified customer experience across Virgin Group's diverse business units. We designed and implemented a complete cross-platform mobile application in just 36 hours that allows users to engage with Virgin services, earn and redeem loyalty points, and manage their bookings.

Flutter Dart Firebase Firebase Auth Cloud Firestore

Key Features

  • Personalized User Profiles: Users can create and customize their profiles with preferences for Virgin services
  • Points System: Earn points through purchases, activities, and social contributions
  • Rewards Redemption: Exchange points for discounts and exclusive offers
  • Ticket Management: View and manage all bookings and tickets in one place
  • QR Code Scanner: Scan codes at physical locations to earn badges and points
  • Service Marketplace: Browse and purchase various Virgin services

Development Process

Our approach focused on creating an intuitive user experience while ensuring robust data management. We prioritized a clean, responsive design following Virgin's brand guidelines. The development process involved:

  1. Initial wireframing and UI design in Figma
  2. Implementing core features using Flutter
  3. Creating responsive layouts that work across different device sizes
  4. Testing and refining the user experience

The most challenging aspect was integrating multiple features while maintaining a cohesive user experience, all within the tight 36-hour timeframe.
